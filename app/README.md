# BeatsPerfect Shop

Static site for `beatsperfect.shop`.

## Run locally

```bash
cd shop/app
python3 -m http.server 4173
```

Then open `http://localhost:4173/`.

## Structure

- `index.html` home
- `catalog/` products browser
- `product/` product detail view
- `about/` how we build page
- `faq/` support page
- `contact/` contact page
- `legal/` legal notes
- `assets/` shared data, styles, and rendering logic
